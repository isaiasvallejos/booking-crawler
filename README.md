# Booking Crawler

## How to run

```sh
$ git clone git@github.com:isaiasvallejos/booking-crawler.git
$ npm install
$ npm run dev:http # run http api on localhost:8080
$ npm run test # for tests
```

## Usage

`POST /api/search`

```ts
checkIn: string // YYYY-MM-DD
checkOut: string // YYYY-MM-DD
```
